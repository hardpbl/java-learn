类变量和局部变量的区别
---

####类变量：
linking的prepare阶段给类变量默认赋值，-->initial阶段给类变量显示的赋值
既静态代码块赋值

####实例变量
    随着对象的创建在堆空间分配实例变量空间，并进行默认赋值
####局部变量
    在使用前必须显示的赋值，否则编译不通过
